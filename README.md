# Blackjack

A command line blackjack game built in Java. Play against a dealer that follows standard Blackjack rules.

## Features 
- Dealer AI that hits until 17
- Bust detection for both player and dealer
- Personalized messages with player name
- Play again option 

## How to Run
```bash
javac Blackjack.java
java Blackjack
```


## Technologies
- Java
